# 📖 [Global Summit 2015] <a name="about-project">About</a>

This website is about the Global Summit 2015 which is planned for TWO days (15-Oct-2015 to 16-Oct-2015) in Korea. This website contains the details of the summit, the activities going to be during the summit, the booking for the summit, the speakers presenting during the summit and also the glimpses of the previous 2 summits.
